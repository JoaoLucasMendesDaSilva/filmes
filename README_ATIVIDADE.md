ATIVIDADE: Integração de Middleware no Redux

Projeto MovieMania feito com React, Redux, Redux Thunk, Redux Saga, Redux Logger e Axios.

Como usar:
1. Crie o projeto:
   npx create-react-app moviemania

2. Entre na pasta:
   cd moviemania

3. Instale as dependências:
   npm install redux react-redux redux-thunk redux-saga redux-logger axios

4. Copie a pasta src deste arquivo ZIP para dentro do projeto, substituindo a pasta src original.

5. Nos arquivos abaixo, troque YOUR_API_KEY pela sua chave da OMDb API:
   src/redux/thunkActions.js
   src/redux/sagas.js

6. Rode:
   npm start

Observação:
A busca pode ser feita pelos botões "Buscar com Thunk" ou "Buscar com Saga".
O redux-logger mostra as ações no console do navegador.
